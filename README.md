# LL1-test
编译原理实验-LL1 (好像有bug)

#### 实验目的
根据某一文法编制调试LL（1）分析程序，以便对任意输入的符号串进行分析。本次实验的目的主要是加深对预测分析LL（1）分析法的理解。

#### 测试案例
简单要求：<br>
至少做到对下列已知的文法，用LL（1）分析法对任意输入的符号串进行分析： <br>
（1）E->TG<br>
（2）G->+TG<br>
（3）G->ε<br>
（4）T->FS<br>
（5）S->*FS<br>
（6）S->ε<br>
（7）F->(E)<br>
（8）F->I<br>
<br>
（注意！用$代替空串，用的char储存）
<br>
高要求：<br>
  1、手动输入文法<br>
  2、显示文法的非终结符的Follow集<br>
  3、显示规则的可选集合<br>
  4、构造预测分析表<br>
  5、对任意输入的符号串进行分析<br>

#### 代码简称
有分析固定文法和分析手动输入的文法两个模块，后面一个有问题，加了消除左递归，但好像有bug
